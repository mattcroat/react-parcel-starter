# ⚛React-📦Parcel 🏁Starter

## Getting Started

📜 Install Parcel

```shell
npm i -g parcel-bundler
```

📜 Clone Repo and Install Dependencies

```shell
git clone https://github.com/mattcroat/react-parcel-starter.git . && npm i
```

🏃‍ Run

```shell
npm start
```

🌍 Visit http://localhost:3000

## Other

📦 Build

```shell
npm build
```

💁‍ Sass Support

```
npm i node-sass
```
